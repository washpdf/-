三根智诊：基于蒙医理论的AI舌脉问辅诊与趋势监测系统
====================================================================

项目简介
--------
本项目是一个基于蒙医三根理论的AI辅助诊疗系统，结合深度学习、知识图谱和大语言模型技术，提供舌诊、问诊、脉诊三诊融合的智能辨证服务。

项目结构
--------
team/
├── backend/          # 后端服务目录
├── frontend/         # 前端页面目录
└── readme.txt        # 本文件

主要功能
--------
1. 舌象分析：基于计算机视觉的舌象识别与分析
2. 智能问诊：引导式对话采集症状信息
3. 脉诊记录：脉象特征采集与分析
4. 三根辨证：三诊融合的蒙医体质辨证
5. 调理方案：基于AI辨证的个性化健康建议
6. 趋势监测：LSTM+注意力机制的健康趋势预测
7. 知识图谱：基于Neo4j的蒙医知识图谱展示

技术栈
------
后端：
- Flask：Web框架
- SQL Server：用户数据存储
- Neo4j：知识图谱数据库
- TensorFlow/Keras：LSTM预测模型
- ONNX Runtime：舌诊模型推理

前端：
- 原生HTML5/CSS3/JavaScript
- Chart.js：数据可视化
- Font Awesome：图标库

代理服务：
- Node.js + Express：阿里云百炼API代理
