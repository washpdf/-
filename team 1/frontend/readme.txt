frontend 目录 - 前端页面
====================================================================

目录作用
--------
本目录包含项目的前端页面代码，提供用户交互界面，包括舌象上传、问诊对话、脉诊记录、辨证报告展示等功能。

文件说明
--------
├── index.html                # 主页面文件，包含完整的前端界面
├── best.onnx                 # 舌诊AI模型（ONNX格式）
└── diagnose_model.py         # 舌诊模型推理脚本

页面功能模块
------------
首页模块：
- 蒙医三根理论介绍
- 项目特色优势展示
- 知识图谱可视化
- 服务内容介绍

诊断流程：
1. 舌象分析页：上传舌象照片，AI自动识别分析
2. 智能问诊页：引导式对话采集症状
3. 脉诊记录页：脉象特征数据采集
4. 三根辨证页：三诊融合辨证报告
5. 调理方案页：个性化健康调理建议
6. 周期检测页：健康趋势监测与预测
7. 蒙医知识页：蒙医理论知识科普

技术实现
--------
- 纯原生HTML5/CSS3/JavaScript，无需框架依赖
- Chart.js：用于趋势图表展示
- Marked.js：Markdown渲染（用于AI生成的调理方案）
- 响应式设计，支持移动端和桌面端

外部资源
--------
- B站视频嵌入：舌象拍摄指导
- CDN资源：Chart.js、Font Awesome、Marked.js

使用说明
--------
直接在浏览器中打开 index.html 即可访问，或通过后端Flask服务访问（推荐）。
